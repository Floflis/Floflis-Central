info Instructions on modifying this site/page/app:

  1. public Language: in the <html lang="en-us"> tag, change en-us to the code of the language used in your project.
  2. color_lens Colors: go to the /site folder in the framework.zip archive, open the index.html and go to the format_paint Personalize heading that is bellow the create Starting heading.
  3. tag_faces Favicons: use your favicon url in <link rel="icon" href="icon.png" /> and your SVG icon url in <link rel="logo" href="icon.svg" border="true" />.
  4. space_bar tab title and his color: for browser that support PEHTML, you can set the color of your site's title in the browser tab. Rename the title and use your color in the <title color="">New Framework.js project</title> tag.
  5. info Description: this improves how your project appears on Google and other search engines. In the tag <meta name="description" content=""> simply describe your project in the content="" attribute.
  6. format_quote Keywords: this improves how your project appears on Google and other search engines. In the tag <meta name="keywords" content=""> write in commas all the words that describes your project in the content="" attribute.
  7. photo_size_select_actual Logo: in the <logo>New Framework.js project</logo> tag, change New Framework.js project to the name of your project. If you want to use a image as the logo, go to the /site folder in the framework.zip archive, open the index.html and go to the format_paint Personalize heading that is bellow the create Starting heading.
  8. comment Comments: If you want to use a commenting system in your project, insert the code where it will appear inside the <blockquote> tag that is inside the <comments> tag. The comment count you can add inside the <span class="badge"></span> that is inside the comment Comments heading.

  9. Last update date: inside the <center> tag that is inside the <footer> tag, write the last update date, that is when you edited this file. You can remove the footer if you just want to make a app in this file.
  10. receipt Badges: togheter the Framework.js and PEHTML's badges, you can put your own badges, such as: AngularJS, HTML5, etc.
  11. tag_faces Social icons: inside the <div class="social-icons"> tag, put your social images with links. You can also add the title attribute to show a tooltip about each social network and his contents.
  12. touch_app Social buttons: inside the <div class="social-buttons"> tag, put your social buttons (for sample: like, +1, tweet, and others).
  13. info Credits: in the <copyright> tag, edit it as you please.
  14. check_circle Done! Personalize and add elements to this as you want.
